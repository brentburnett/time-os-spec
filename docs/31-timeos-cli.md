# TIME-OS Unified Developer Specification
## Section 31 — TIME-OS CLI

### 31.1 Purpose
Defines the command-line interface for interacting with TIME-OS engines, packs, nodes, and TMV.

### 31.2 CLI Fundamentals
CLI provides:
- Engine execution
- Pack management
- TMV queries
- Routing previews
- Node diagnostics

### 31.3 Core Commands
- timeos engine run
- timeos pack join
- timeos tmv get
- timeos routing preview
- timeos node status

### 31.4 Deterministic CLI Rules
CLI output must match API output exactly.

## End of Section 31 — TIME-OS CLI
