# TIME-OS Unified Developer Specification
## Section 36 — Mobile Runtime

### 36.1 Purpose
Defines TIME-OS runtime behavior on mobile devices.

### 36.2 Mobile Capabilities
Mobile runtime supports:
- Local routing preview
- TMV queries
- Pack membership
- Engine sandboxing

### 36.3 Mobile Constraints
Mobile nodes cannot host full engine clusters.

## End of Section 36 — Mobile Runtime
